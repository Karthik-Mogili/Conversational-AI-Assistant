# AI Chatbot using Groq API

A professional AI chatbot built using:

- Python
- Streamlit
- Groq API
- Llama 3 Models

## Features

- Conversational AI Chatbot
- Streaming Responses
- Chat History
- Model Selection
- Clean Streamlit UI
- Prompt Engineering Support

## Technologies Used

- Python
- Streamlit
- Groq API
- OpenAI SDK

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
streamlit run app.py
```

## Models Used

- llama-3.3-70b-versatile
- llama3-8b-8192
